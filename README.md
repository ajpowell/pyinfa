# pyinfa
Python class to make PowerCenter command line tools easier to use.

Curently supports:

## pmrep
- pmrep version
- pmrep connect
- pmrep listfolders
- pmrep listworkfolders

## pmcmd
- pmcmd ping
- pmcmd startworkflow

## infacmd
- infacmd listservices
- infacmd listnodes
- infacmd listlicenses
- infacmd showlicense
